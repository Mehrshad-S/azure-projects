# Azure Projects

This repository contains my hands-on Microsoft Azure projects focused on cloud administration, networking, identity, security, backup, monitoring, and AZ-104 preparation.

I built these projects through self-study, lab environments, Azure deployments, and certification preparation to document my practical experience with Azure services and cloud infrastructure.

## Projects

| Project                                                                            | Description                                                                                                                                   |
| ---------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| [AZ-104 Hands-On Lab Portfolio](./az104-hands-on-lab-portfolio)                    | Full Azure Administrator lab portfolio covering identity, RBAC, governance, networking, storage, compute, containers, backup, and monitoring. |
| [Azure Secure VPN & Cloud Backup Solution](./secure-vpn-cloud-backup)              | Secure remote access solution using Azure VPN Gateway, Azure File Share, mapped network drives, and backup policies.                          |
| [Azure Multi-Region Website Deployment](./multi-region-website-load-balancing)     | Multi-region website architecture with regional load balancing and global traffic routing.                                                    |
| [Azure Internal Load Balancer & Private DNS](./internal-load-balancer-private-dns) | Internal web architecture using IIS servers, private DNS, VNet peering, Bastion, and an internal load balancer.                               |
| [Azure AD DS & Multi-VNet Infrastructure](./ad-ds-multi-vnet-infrastructure)       | Multi-VNet Azure infrastructure with AD DS, domain-joined VMs, internal DNS, IIS, and Azure Bastion.                                          |
