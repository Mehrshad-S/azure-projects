# Azure Secure VPN & Cloud Backup Solution

This project focused on building a secure remote access and cloud backup solution in Microsoft Azure.

The goal was to allow remote users to connect securely into the Azure environment, access shared files through Azure Files, and protect important data using automated backup policies.

## Project Date

Nov 2025 – Dec 2025

## What I Implemented

* Set up a secure Point-to-Site VPN connection for remote access into the Azure environment.
* Deployed an Azure File Share for centralized file storage.
* Mapped the Azure File Share to a virtual machine as a network drive.
* Configured automated backup policies with weekly and monthly schedules.
* Defined backup retention periods for short-term and longer-term recovery.
* Tested the solution by uploading files, running manual backups, and restoring data into a recovery folder.

## Technologies Used

* Azure VPN Gateway
* Point-to-Site VPN
* Azure Files
* Azure Storage Account
* Azure Backup
* Recovery Services Vault
* Virtual Machines
* Networking and remote access
