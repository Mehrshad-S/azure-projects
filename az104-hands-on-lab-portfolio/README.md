# AZ-104 Hands-On Lab Portfolio

This project is my Microsoft Azure Administrator hands-on lab portfolio completed as part of my AZ-104 certification preparation.

The portfolio documents multiple Azure administrator labs covering identity, access management, governance, networking, storage, compute, containers, backup, monitoring, and alerting.

## What I Completed

- Microsoft Entra ID user and group management
- Azure subscriptions, RBAC, and custom roles
- Azure Policy, tags, remediation, and resource locks
- ARM templates and Bicep deployments
- Virtual networks, subnets, NSGs, ASGs, and DNS zones
- VNet peering and custom route tables
- Azure Load Balancer and Application Gateway
- Azure Storage, Blob containers, Azure Files, and SAS access
- Virtual Machines and Virtual Machine Scale Sets
- Azure App Service, deployment slots, and autoscaling
- Azure Container Instances and Azure Container Apps
- Azure Backup and Site Recovery
- Azure Monitor alerts, action groups, and alert processing rules
